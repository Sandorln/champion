# LOL CHAMPION

라이엇에서 제공하는 API를 통해,
최신 버전 ~ 이전 버전의 챔피언, 아이템, 소환사 주문 정보를 확인할 수 있습니다.

## Google Play Store
https://play.google.com/store/apps/details?id=com.sandorln.champion

## 개발 환경
- 언어 : Kotlin
- 아키텍쳐 : MVI
- 라이브러리 : Compose, Coroutine, Hilt, Ktor, Glide, Exoplayer, Motion Layout, Room
- SDK : Min / Target 34
